# 01 - Java 17+

## 🎯 Objetivo

Entender os recursos modernos da linguagem Java usados no mercado e criar uma base confortável para projetos backend.

## ✅ Checklist

- [ ] Diferença entre versões LTS e não LTS
- [ ] Sintaxe básica
- [ ] Classes
- [ ] Métodos
- [ ] Pacotes
- [ ] Modificadores de acesso
- [ ] `var`
- [ ] `switch expressions`
- [ ] `text blocks`
- [ ] `records`
- [ ] `sealed classes`, apenas noção inicial
- [ ] Boas práticas com Java moderno

## 💡 Por que importa no backend?

Java 17 é uma versão LTS muito usada em aplicações corporativas e projetos Spring Boot modernos.

## 🧪 Exercício sugerido

Criar pequenos exemplos usando classes, records, enums, switch expressions, text blocks e `var`.
