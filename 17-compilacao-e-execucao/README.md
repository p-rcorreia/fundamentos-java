# 17 - Compilação e execução

## 🎯 Objetivo

Entender o caminho do código-fonte Java até a aplicação rodando.

## ✅ Checklist

- [ ] Arquivo `.java`
- [ ] Compilador `javac`
- [ ] Bytecode `.class`
- [ ] Execução com `java`
- [ ] Classpath
- [ ] JAR
- [ ] Fat JAR
- [ ] Build com Maven
- [ ] Build com Gradle, noção
- [ ] Compilação vs empacotamento
- [ ] Tempo de compilação vs tempo de execução
- [ ] Erros de compilação
- [ ] Erros de runtime
- [ ] Como o Spring Boot inicia

## 💡 Por que importa no backend?

Todo projeto backend precisa ser compilado, empacotado, executado e entregue em algum ambiente.

## 🧪 Exercício sugerido

Criar uma classe simples, compilar com `javac`, executar com `java` e depois empacotar em JAR.
