# ☕ Fundamentos Java

Repositório dedicado aos meus estudos de **Fundamentos Java**, seguindo a primeira etapa do meu **Java Backend Developer Roadmap 2026**.

A ideia é estudar a base da linguagem com calma, registrar percepções importantes e criar exemplos pequenos antes de avançar para Spring Boot, banco de dados, arquitetura, segurança e produção.

---

## 🎯 Objetivo

Construir uma base sólida em Java para desenvolvimento backend profissional.

Este repositório não é uma coleção de anotações soltas. Cada tópico deve conectar:

```txt
conceito -> exemplo -> percepção -> armadilha -> uso no backend
```

## 🧪 Estrutura de cada tópico

Cada pasta começa simples:

```txt
topico/
├── README.md
└── exemplos/
```

O README de cada tópico deve evoluir com:

- explicação simples;
- checklist de subtópicos;
- exemplos pequenos;
- armadilhas comuns;
- relação com backend Java;
- exercício prático;
- percepções pessoais.

---

## 📝 Convenção de commits

- 📁 `estrutura:` criação ou organização de pastas e arquivos
- 🧠 `conceito:` explicações e percepções teóricas
- ✅ `exemplo:` código prático para fixar o conteúdo
- ⚠️ `armadilha:` registro de erro comum ou ponto de atenção
- 🔁 `refatoracao:` melhoria de exemplos ou explicações

Exemplos:

```txt
📁 estrutura: adiciona trilha de fundamentos Java
🧠 java-17: registra conceitos de LTS e recursos modernos
✅ java-17: adiciona exemplos com records e switch expressions
⚠️ optional: registra uso indevido em atributos de entidade
```
